Problem 1: Monthly Budget Analyzer

Pehle program me maine student ke monthly expenses ko alag alag variables me store kiya jaise rent, food, transport aur utilities. Us ke baad maine in sab ko add karke total monthly expense calculate kiya. Phir maine ek budget limit set ki jo 60,000 PKR thi. Us ke baad maine if-else condition lagayi jisme check kiya ke agar total expense budget limit se kam ya barabar hai to student within budget hai, warna over budget hai. Is program me maine variables, addition aur simple conditional logic use ki.

Electricity Bill Estimator

Dusre program me maine user se electricity units ka input liya aur us ko integer me convert kiya. Phir maine if-elif-else structure use kiya taake pata chal sake ke units kis category me aate hain. Agar units 100 ya us se kam the to basic rate apply hota hai. Agar units 101 se 300 ke darmiyan the to standard rate apply hota hai. Aur agar units 300 se zyada the to high usage rate apply hota hai. Is program me maine user input, multiple conditions aur logical decision making use ki.

Class Attendance Analysis

Is program me maine attendance percentages ki ek list banayi. Phir maine do counters banaye, ek un students ke liye jo 75% ya us se zyada attendance rakhte hain aur doosra un ke liye jo requirement puri nahi karte. Phir maine for loop lagaya jo list ke har percentage ko check karta hai. Agar percentage 75 ya us se zyada hota hai to meet requirement counter increase hota hai, warna doosra counter increase hota hai. Aakhir me maine dono totals print kar diye. Is program me maine loop, counter aur conditional logic ka use kiya.

Problem 4: Scholarship Eligibility System

Is program me maine do lists use ki, ek students ke CGPA ke liye aur doosri un ki family income ke liye. Phir maine loop chalaya jo har student ka CGPA aur income ek sath check karta hai. Maine compound condition use ki jisme check kiya ke agar CGPA 3.5 ya us se zyada ho aur family income 80,000 ya us se kam ho to student scholarship ke liye eligible hai. Agar dono conditions true hoti hain to eligible students ka counter increase hota hai. Is tarah maine real-life scholarship rule ko program me apply kiya.

Problem 5: Sensor Alert System

Is program me maine sensor readings ki ek list banayi aur ek safe range define ki jo 20 se 80 ke darmiyan thi. Phir maine loop lagaya jo har reading ko check karta hai. Agar koi value safe range se bahar hoti hai to alert counter increase hota hai. Us ke baad maine total readings nikali aur unsafe readings ka percentage calculate kiya. Is program me maine accumulator use kiya, logical checking ki aur percentage calculate ki.
