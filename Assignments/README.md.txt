Problem 1: Monthly Budget Analyzer
Is problem mein aapne basic Variables aur Arithmetic Operations ka istemal kiya hai. Aapne kharchon (expenses) ko jama kar ke total nikala aur phir Simple If-Else condition ke zariye ye check kiya ke kya kharcha 60,000 ki limit se zyada hai ya kam. Ye program sikhata hai ke kaise real-world data ko programming logic mein convert karte hain.

Problem 2: Electricity Bill Estimator
Yahan aapne User Input aur Multiple Conditions (if-elif-else) ka concept apply kiya hai. Bijli ke units ke hisab se mukhtalif slabs (categories) banana coding mein "decision making" ka ahem hissa hai. Is se ye pata chalta hai ke programming mein range-based logic kaise kaam karti hai.

Problem 3: Class Attendance Analysis
Is sawal mein aapne Lists aur Loops ka istemal kiya hai. Aapne ek list ke andar maujood har value ko baari baari (iterate) check kiya aur Counters ka use karte hue ye ginna (count) ke kitne students 75% ki requirement poori kar rahe hain. Ye loop aur logic ka behtareen combination hai.

Problem 4: Scholarship Eligibility System
Ye thora advanced level ka kaam hai kyunke is mein Compound Conditions (AND operator) ka istemal hua hai. Aapne do mukhtalif filters (CGPA aur Income) ko ek sath check kiya. Is se ye samajh aata hai ke jab aik se zyada sharait (conditions) poori karni hon, to unhein logic mein kaise merge kiya jata hai.

Problem 5: Sensor Alert System
Is bonus problem mein aapne Data Analysis ki basic seekhi hai. Safe range (20-80) se bahar ki values ko pehchan-na aur phir unka Percentage nikalna ye dikhata hai ke aap data se kaam ki maloomat (insights) nikalna seekh chuki hain. Is mein "Accumulators" aur mathematical calculation dono ka istemal hua hai.
